Command line app to interact with Medium.

## Setup

Install virtualenv (may require `sudo`)
```bash
$ pip install virtualenv
$ virtualenv venv
$ . venv/bin/activate
$ pip install -r requirements.txt
$ python setup.py develop
```

Give it a try!
```bash
$ . venv/bin/activate
$ dragon
```
